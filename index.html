Why this project exists

What this demonstrates

This project simulates a small marketing analytics system that tracks customer segments, campaign sends, engagement, purchases, and revenue. The GitHub Pages version is static, which means it works with plain HTML, CSS, and JavaScript and needs no backend.

Good interview explanation

I built a customer campaign analytics project to show how a company could measure marketing performance, customer engagement, and purchase behavior, then turned the results into a clean dashboard for business users.

GitHub Pages

This package is ready for GitHub Pages. Upload these files to a repository, enable Pages, and it will work as a static site. The .nojekyll file is included to avoid build issues.
